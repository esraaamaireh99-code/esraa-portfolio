# Esraa Amaireh Portfolio

Modern CV and portfolio website for Esraa Amaireh, a Full Stack Developer focused on Laravel, React, Next.js, Kotlin, Spring Boot, databases, and API integrations.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- React

## Features

- Responsive portfolio homepage with hero CTAs, technology badges, featured experience, skills, projects, and contact links.
- Redesigned resume page with structured CV sections, sticky download action, experience cards, grouped skill pills, education, and certifications.
- Data-driven content from `src/data/portfolio.ts`.
- SEO and Open Graph metadata.
- Downloadable CV PDF from `public/Esraa_Amaireh_CV.pdf`.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

Run production checks:

```bash
npm run lint
npm run build
```

## Deploy on Vercel

Import the GitHub repository into Vercel and deploy with the default Next.js settings. The production build command is `npm run build`.
